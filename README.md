# designStack

**Claude Code for designers building their own products.**

---

## Install

Open your terminal and paste:

```bash
curl -fsSL https://raw.githubusercontent.com/harshii0509/designStack/main/install | bash
```

Then restart Claude Code. Type `/ds:start`.

That's it. The wizard takes 5 minutes.

---

## What it does

| Command | What it does |
|---------|--------------|
| `/ds:start` | First-time setup. Run once per project. |
| `/ds:plain` | Translates any Claude plan to plain English before you say yes |
| `/ds:unstuck` | Something broke — one sentence diagnosis, one fix |
| `/ds:save` | Restore point. Like Save in a game. |
| `/ds:look` | Does it look right? Scored against your brand rules |
| `/ds:mobile` | Phone, tablet, desktop — side by side |
| `/ds:a11y` | Accessibility grade A–D, every issue labeled |
| `/ds:context` | Build your Design Bible once. Claude knows your brand forever. |
| `/ds:share` | Shareable link in under 2 minutes |
| `/ds:vibe` | Describe how it should feel — get 3 directions to pick from |
| `/ds:brand` | Full-site brand drift check |
| `/ds:polish` | Pre-ship quality check — 11 areas |
| `/ds:animate` | Add purposeful motion |
| `/ds:delight` | Add joy to the moments that matter |
| `/ds:stats` | See your usage and success rates |

---

## How it works

Run `/ds:start` once in your project. It asks 3 questions and builds your **Design Bible** — a file at `design/DESIGN-BIBLE.md` that stores your brand colors, fonts, and rules.

Every skill reads from it automatically. You never explain your brand to Claude again.

## Works with
